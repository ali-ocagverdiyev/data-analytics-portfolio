# Healthcare Data Analysis Project

This project analyzes synthetic hospital data using R and R Markdown.

The analysis focuses on descriptive statistics, anomaly detection, and risk-related patterns in patient-level healthcare records.

## Project Overview

The project was completed as a final data analytics project.  
The dataset includes patient information, clinical observations, and encounter records.

## Main Topics

- Descriptive analysis
- Data quality checks
- Anomaly detection
- Patient-level risk signals
- Healthcare data visualization
- R Markdown reporting
- Shiny dashboard development

## Tools Used

- R
- RStudio
- R Markdown
- Shiny
- ggplot2
- dplyr
- data.table
